Selfiebot - Microcontrollers Project

System that uses facial detection to track faces and take pictures of people. The camera on the robot pans and tilts using servos controlled from an ATmega1284 microcontroller. A laptop computer performs image processing and tracking, using OpenCV for facial detection, giving commands to the microcontroller via serial communication.

See documentation for more details.
