Okra - Check Splitting App

- Architected and implemented a mobile web app using Flask (Python) as a web framework - Designed and implemented responsive UI optimized for the phone (js, html/css) - Collaborated with a team of 5 engineers for a hackathon Okra is an app designed to make splitting a check a breeze, guiding the user through a step by step process from receiving the receipt to paying back. Okra can also work with any kind of purchase. - The user creates a tab and invites friends (Facebook or Venmo) - The app uses OCR to parse and itemize a receipt, and allows the user to select which items each person bought. If there is no receipt the user may manually input fields. - The app then splits the prices including tip and tax, and allows each the user to pay back the primary tab creator via Venmo. We are currently working to implement Okra in iOS, and release it in the near future. We are also looking to implement additional features for payback beyond Venmo and build more options for splitting. Github: https://github.com/sudov/okra
